```
├── package.json

```

## Setup Instructions
1. Install dependencies:
   ```bash
   npm install
   ```

2. Configure environment variables in `config/.env`

3. Start the server:
   ```bash
   npm start
   ```

## Features
- User Authentication
- Venue Booking System
- Admin Dashboard
- Owner Dashboard
- Real-time Availability
- Email Notifications
